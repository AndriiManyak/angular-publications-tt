# AngularSmidaTt

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.0.3.

## Local deployment

### Dependencies
* Node v14.7.0
* NPM v6.14.7

### Installing
* Fork and clone this repository
* Run `npm install` in your terminal
* Run `npm run server`
* Run `npm start`
